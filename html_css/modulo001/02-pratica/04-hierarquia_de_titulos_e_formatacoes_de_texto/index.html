Título de nível 3

Como muitos podem pensar, a hierarquia de títulos não serve apenas para mudar o tamanho das letras mas, é algo muito mais importante pois, cada nível de título serve para uma determinada ocasião.

É legal dizer que, dentro de um título de nível 2 podem ter vários títulos de nível 3, e todos esses títulos ficam dentro do título de nível 1.

Principais Formatações

Negrito.. ou, destaque

Nesta frase temos um termo em negrito usando a tag "b".

Já nesta frase, temos um termo de destaque usando a tag "strong".

___

Itálico.. ou, ênfase

Nesta frase, temos um termo em itálico usando a tag "i".

Já nesta frase, temos um termo em ênfase usando a tag "em".

Texto marcado

Podemos criar também um texto marcado usando a tag "mark".

Usando essa mesma tag neste parágrafo, temos outro texto marcado no final.

Texto grande e pequeno

Estamos criando um "TEXTO GRANDE" usando a tag "big" e um "TEXTO PEQUENO" usando a tag "small", ambos no mesmo parágrafo.

Texto deletado

Podemos marcar um texto como excluído usando a tag "del" para indicarmos que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido usando a tag "ins" para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado com a tag "u".

Texto sobrescrito

Para inserir coisas do tipo x20+3, usamos a tag "sup" no valor que será elevado.

Texto subscrito

Para inserir coisas do tipo h2O, usamos a tag "sub" no valor que será abaixado.
